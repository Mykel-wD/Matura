git log - commits

					Workflow:
	1.	git pull - to get the latest changes from the remote repository
 	2.	Make changes to your local copy of the code
	3.	Wszystko działa:
			(if new files addes): git add - to stage your changes
			git commit - to save your changes locally
			git push - to push to github
	3.	Nie działa:
			reverse changes (git restore files / git checkout {wersja comitu})
	git pull - everytime someone uploads stuff
